.about

Το gnu.gr είναι ένας public jabber server. Το Jabber (ή αλλιώς xmpp) είναι ανοιχτό πρωτόκολλο επικοινωνίας πραγματικού χρόνου. Σε αντίθεση με τα περισσότερα πρωτόκολλα, βασίζεται σε αποκεντρωμένη αρχιτεκτονική. Αυτό σημαίνει πως δεν υπάρχει κάποιος κεντρικός server, αλλά το δίκτυο περιλαμβάνει πολλούς ισότιμους servers. Σε μεγάλο βαθμό θυμίζει το email. Κάθε χρήστης μπορεί ελεύθερα να φτιάξει λογαριασμό σε όποιον server θέλει (και εμπιστεύεται) για να συνομιλεί με όλους του χρήστες του δικτύου. Περισσότερες τεχνικές λεπτομέρειες υπάρχουν στο skytal.es.

.ασφάλεια

Η επικοινωνία των παραπάνω εφαρμογών με το gnu.gr γίνεται κρυπτογραφημένα, όπως ακριβώς και η επικοινωνία με τους υπόλοιπους jabber servers.

Για κρυπτογραφία από άκρη σε άκρη (end to end) συνίσταται η χρήση του πρωτοκόλου OTR (Off-the-Record). Όλες οι προτεινόμενες παραπάνω εφαρμογές το υποστηρίζουν, είτε εγγενώς είτε μέσω κάποιου plugin.

.privacy

Εν μέρει το κίνητρο για τη δημιουργία του gnu.gr ήταν να φτιαχτεί ένας jabber server που θα παρακρατά τον ελάχιστο δυνατό όγκο (μετα)δεδομένων και μόνο αυτών που είναι απαραίτητα για τη λειτουργία του, με σεβασμό στην ιδιωτικότητα των χρηστών.

Δεδομένα που δεν κρατάει ο server.

  • Μεταδεδομένα σύνδεσης. Ακριβή χρόνο και διάρκεια σύνδεσης με την υπηρεσία.
  • Στοιχεία πρόσβασης (access logs). Τόσο στον jabber server όσο και στον web server που φιλοξενεί το παρόν site.
  • Δεδομένα συνομιλιών. Μπορεί να είναι προφανές, αλλά είναι σημαντικό να τονιστεί πως ο server δεν έχει γνώση για το περιεχόμενο των συνομιλιών.

Για τη λειτουργία της υπηρεσίας είναι απαραίτητη η διατήρηση των παρακάτω δεδομένων.

  • Οι επαφές των χρηστών (roster). Αυτό εξασφαλίζει πως η λίστα επαφών παραμένει η ίδια για κάθε account ανεξαρτήτως ποια εφαρμογή ή συσκευή χρησιμοποιείται για τη σύνδεση.
  • Offline μηνύματα. Αυτά είναι εισερχόμενα μηνύματα που αποθηκεύονται στην περίπτωση που ο χρήστης είναι offline. Τα μηνύματα αυτά διαγράφονται όταν ο χρήστης συνδεθεί.
  • Κρατάμε μια εγγραφή, σε επίπεδο μήνα, για την τελευταία φορά σύνδεσης στον server. Με αυτό τον τρόπο διαγράφουμε λογαριασμούς που είναι ανενεργοί πάνω από ένα χρόνο, ώστε να μην κρατάμε στοιχεία χρηστών που δεν χρησιμοποιούν πια την υπηρεσία ή έχουν ξεχάσει τον κωδικό τους.
